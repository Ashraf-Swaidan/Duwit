# Landing page images

**Copy-paste prompts for generative AI:** see **[IMAGE_PROMPTS.md](./IMAGE_PROMPTS.md)** (detailed hero + steps prompts, negatives, and consistency notes).

Drop your AI-generated (or other) assets here and reference them from the app.

| File        | Suggested use                                      |
| ----------- | -------------------------------------------------- |
| `hero.webp` | Main hero visual (right side / below on mobile).  |
| `steps.webp`| Optional illustration for “How it works”.         |

Recommended: **WebP**, reasonable width (~1200–1600px for hero), compressed.

The UI falls back to a built-in gradient composition if a file is missing or fails to load.
